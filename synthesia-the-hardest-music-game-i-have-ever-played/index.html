Synthesia - The hardest musical game I have ever played

Chenglin Lu

May 17, 2020 • 2 min read

Installation

I'm using Fedora 30. The installation is straight forward with Wine (I'm using wine.x86_64 5.0-1.fc30).

Btw don't try playonlinux although suggested by this post (reddit). I tried, installed playonlinux, installed a python 2 version dedicated for playonlinux, installed wxPython Phoenix 4.1 package, tried to fix some playonlinux souce code but there are still lots of bugs. If you still wanna try playonlinux, probably you should use wxPython 2 instead of wxPython Phoenix.

Setup

Yamaha PSR-EW300 SA 76-Key Portable Keyboard

  • + USB 2.0 "Printer" cable (A-Male to B-Male, also see https://synthesiagame.com/keyboards/Help/desktop/usb). The keyboard has the B-Female with words "USB to Host" next to it.
  • + (Optional) Lightning to USB Adapter for my iPad.
  • + 1/4 Inch Stereo Male to 3.5mm Stereo Female Adapter for my headphones. The keyboard has the 1/4 Inch Stereo Female.
  • + 6 AA Rechargeable Batteries.
  • + Yamaha PA130 120 Volt Keyboard AC Power Adaptor. It also works with the power adaptor from my Seagate external hard drive.

I started with the iPad Synthesia app. It's good for playing included music pack and have the same experience of adding external midi file. However it does not support importing .synthesia metadata (github) which I used to add finger hints for external songs (most pre-installed music already has the finger hints). As a result, I also purchased the Synthesia laptop application.

And then.. Practice!
